# Jungle

Git Repository - COMP3211 Software Engineering Group Project "Jungle" - 2022 Fall

## How to run

```shell
mvn clean package appassembler:assemble
```

### Unix

For Mac OS X, it is highly recommended to use the default terminal.

#### Run application

```shell
./run
```

#### Rebuild and Run

```shell
./run rebuild
```
